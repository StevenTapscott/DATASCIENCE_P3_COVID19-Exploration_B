# Project 3: COVID-19 Data Exploration ● ○ ○

## Overview

The Our World in Data COVID dataset is one of the most complete public health datasets ever assembled with 60+ variables covering cases, deaths, vaccinations, and testing across 200+ countries.

This project has you building a visualization-heavy EDA that surfaces real patterns from real data.

---

## Skills you’ll practice

- pandas
- matplotlib
- seaborn
- time series filtering
- data aggregation

---

## Dataset

- Our World in Data COVID-19

---

## Step-by-Step Instructions

1. Download and load the dataset to explore the column list carefully.
2. Filter to a set of countries you want to compare.
3. Visualize case trends over time using line charts.
4. Calculate death rates and vaccination rates per 100,000 people.
5. Create a multi-country comparison chart.
6. Write a brief summary of your findings.

---

## Take it further

Add a rolling 7-day average to smooth out reporting noise.

Then annotate key events (lockdowns, vaccine rollout dates) directly on the chart, annotated time series are far more readable and show you know how to communicate with data.

---

## Why employers care

Working with large, messy public datasets and producing clean, annotated visualizations is a genuine job skill in public health, policy, journalism, and any data role that interfaces with non-technical stakeholders.
